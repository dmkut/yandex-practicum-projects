# Определение выгодного тарифа для телеком компании

**инструменты** 

Python,
Pandas,
Matplotlib,
NumPy,
SciPy,
описательная статистика,
проверка статистических гипотез

**Описание проекта**
Клиентам предлагают два тарифных плана: «Смарт» и «Ультра». 
Чтобы скорректировать рекламный бюджет, коммерческий департамент хочет понять, какой тариф приносит больше денег.

Нужно проанализировать поведение клиентов и сделать вывод — какой тариф лучше.

**Также проверены гипотезы:**

- что средняя выручка с пользователей тарифов «Ультра» и «Смарт» различается. 
    
    p-value 4.26. Отвергли нулевую гипотезу

- что средняя выручка с пользователей из Москвы отличается от выручки с пользователей других регионов. 
    
    p-value 0.525. Не получилось отвергнуть нулевую гипотезу
