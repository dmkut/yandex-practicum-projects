# Определение возраста покупателей

**Инструменты:**

Python, Keras

Нам необходимо построить модель, которая по фотографии определит приблизительный возраст человека. В нашем распоряжении набор фотографий людей с указанием возраста.

**В процессе анализа данных были сделаны следующие выводы:**

1. Для обработки дано 7591 изображений, классы записаны в отдельный файл, ввиду чего необходио использовать параметр flow_from_dataframe.
2. Некоторые изображения аугментированы: Сдвинуты по вертикали/горизонтали, тыкже использован поворот.

По итогам обучения, была достигнута ошибка в 6.57 MAE, что немного превышает требования к задаче. Для обучения была использована архитектура ResNet50, данных в датасете достaточно, чтобы модель не переобучилась. Веса взяты от ImageNet.

**Работа выстраивается в несколько этапов:**

Исследовательский анализ данных:
- Изучение выборки.
- Изучение распределения возрастов в выборке.
- Оценка изображений в датасете.

Обучение модели с использованием ResNet50:

- Формирование обучающей и тестовой выборки.
- Составление функции модели.
- Обучение и оценка МАЕ модели.
