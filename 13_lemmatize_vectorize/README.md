# Обучение модели классификации комментариев

**Инструменты:** Python, Pandas, nltk, tf-idf

Задача проекта: Обучить модель классифицировать комментарии на позитивные и негативные. В нашем распоряжении набор данных с разметкой о токсичности правок.

Метрика качества - F1 не меньше 0.75.

По итогу тексты комментариев были очищены от лишних символов, лемматизированы и векторизированы. Были обучены две модели: логистическая регрессия и LGBM. Первая показала лучший результат при сбалансированном параметре весов классов, на тестовой выборке она показала результат 0.82, что соответствует условию задачи.
